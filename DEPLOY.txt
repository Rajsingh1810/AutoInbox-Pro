🎯 QUICKEST WAY TO DEPLOY (3 STEPS)
================================================================================

STEP 1: Install huggingface_hub
────────────────────────────────
pip install huggingface_hub


STEP 2: Get Your HF Token
────────────────────────────────
1. Go to: https://huggingface.co/settings/tokens
2. Click "New token"
3. Name: "AutoInbox Deploy"
4. Type: "Write"
5. Generate and COPY the token


STEP 3: Run Upload Script
────────────────────────────────
python upload_to_hf.py

When prompted, paste your token.
Wait for upload to complete.

🐛 TROUBLESHOOTING
================================================================================

Problem: Space won't build
Solution: Check "Build Logs" tab for errors

Problem: Import errors
Solution: Verify all packages in requirements.txt

Problem: Models not loading
Solution: Use Git LFS or upload script (handles automatically)

Problem: Blank page
Solution: Wait longer, then try "Reboot" in Settings
